DNA Base Pairing

DNA is comprised of the nucleotide bases adenine (A), thymine (T), cytosine (C), and guanine (G) linked by a phosphate backbone. In double stranded DNA, A and T form linkages with 2 hydrogen bonds, whereas G and C form linkages with 3 hydrogen bonds. Determining the Number of Atoms of Each Element in a DNA Base Pair for Molar Mass Determination

DNA is a macromolecule consisting of multiple nucleotides. Nucleotides are comprised of a nucleoside linked to a phosphate. Nucleosides are a nitrogenous base like a purine (A or G) or a pyrimidine (T or C) linked to a sugar, which in the case of DNA is the pentose sugar deoxyribose. Hence, there are 3 subcomponents of DNA:

  1. Phosphate
  2. Pentose Sugar Deoxyribose
  3. Nitrogenous Bases Adenine, Thymine, Cytosine, and Guanine

Whereas each nucleotide in DNA contains a phosphate and deoxyribose, the nitrogenous bases can vary. Indeed, it is the sequence of nitrogenous bases in the nucleotides along the length of a genome that determine the unique coding can the percentage of each nitrogenous bases vary

Adenine

I will obtain the number of atoms of each element in the Adenine strucutre by parsing its PDB file.

library(tidyverse)

# Read the adenine PDB file into a tibble, skipping the first row
# that causes the column of interest to be missed.
adenine <- read_table2(file = "../../static/files/adenine.pdb", 
                       col_names = FALSE, 
                       skip = 1)

# Remove the last row that only contains "end".
A.atoms <- adenine[seq(1,(nrow(adenine)-1)),12]

# Make a table with the total number of atoms of each type.
A.table <- table(A.atoms)
A.table
## A.atoms
## C H N 
## 5 5 5

The results corroborate with the C5N5H5 chemical formula for Adenine on Wikipedia.

Next, I will calculate the total molar mass of the Adenine molecule. To do this, I will need the atomic mass of each element:

Element Atomic Mass
C 12.0107
N 14.0067
H 1.00794

I will obtain these masses from the R package PeriodicTable.

library(PeriodicTable)

# Change to a data frame.
A.df <- as.data.frame(A.table)

# Add a 3rd column that is the atomic mass of each element.
A.df2 <- cbind(A.df, MW = PeriodicTable::mass(A.df[1:nrow(A.df),1]))

# Add a 4th column that is the total molecular weight of each element
# in the nucleotide base molecule. This is the product of the atomic
# mass and the number of atoms of each element.
A.df3 <- cbind(A.df2, MW.tot = A.df2[,2] * A.df2[,3])

A.df3
##   A.atoms Freq       MW  MW.tot
## 1       C    5 12.01070 60.0535
## 2       H    5  1.00794  5.0397
## 3       N    5 14.00670 70.0335
sum(A.df3[,"MW.tot"])
## [1] 135.1267

Hence, the molar mass of adenine is determined to be 135.1267, which corroborates with the molar mass of 135.13 g/mol listed for Adenine on Wikipedia.
